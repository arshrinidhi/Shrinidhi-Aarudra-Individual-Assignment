# Week 5 – Individual Assignment 2

This repository contains the deliverables for Individual Assignment 2 for Week 5 of the course. The assignment focuses on knowledge graph theory and a comparative review of database systems relevant to building modern knowledge bases.

## Repository Structure

- `part1-kg-readings-summary.md` – Summary and analysis of assigned Knowledge Graphs textbook readings
- `part2-memgraph-review.md` – Review of Memgraph as a graph database
- `part3-timescaledb-review.md` – Review of TimescaleDB and its applicability to knowledge bases
- `part4-paradedb-review.md` – Review of ParadeDB as a PostgreSQL search extension
- `part5-pgvector-review.md` – Review of pgvector with Python and Go support

## Citation Style
